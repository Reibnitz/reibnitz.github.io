O objetivo desse jogo é acertar a palavra secreta dentro de 6 palpites.

Cada palpite precisa ser uma palavra válida de cinco letras. Pressione Enter para verificar o palpite.

Após o palpite ser verificado, as cores das caixinhas irão mudar conforme o exemplo abaixo:

A cor amarela indica que a palavra secreta contém essa letra, porém não nessa posição.

A cor verde indica que a palavra secreta contém essa letra nessa mesma posição.

A cor cinza indica que a palavra secreta não contém essa letra.

Acentuação e Ç são colocadas automaticamente caso façam parte do palpite. Isso não indica que a palavra secreta contém esse acento.

A cada dia, uma nova palavra secreta será sorteada.
